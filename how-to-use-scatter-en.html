如何安装使用Scatter

英文版请参考：https://www.myeoskit.com/#/guides/how-to-set-up-scatter

Scatter是什么？

Scatter是一个浏览器扩展插件，允许你为EOS等区块链签署交易，并向Web应用程序提供个人信息，而无需暴露密钥或填写表单。

——Scatter Github

安装Scatter

第一步：下载Scatter

访问 get-scatter.com or scatter-eos.com，并下载chrome浏览器插件：

第二步：验证Scatter

通过chrome浏览器的开发者选项信息验证Scatter。确保你使用的是官方版本：

  • 在浏览器地址输入chrome://extensions/
  • 打开右上角的开发者模式
  • 验证Scatter的ID: ammjpmhgckkpcamddpolhchgomcojkle

第三步：创建Scatter账户

单击Chrome插件部分中的Scatter图标。 创建将用于访问你的帐户的密码，然后单击“Create New Scatter”。

在此步骤之后，立即记下助记词并将其存储在安全的地方。 在账号意外丢失或被盗的情况下可以通过助记词找回。

⚠️: 如果你觉得使用Scatter感觉不舒服，你也可以使用其他方法，比如cleos命令行工具。 Scatter目前是我们建议的工具之一，因为我们已经验证了它，而且我们知道创始人。

创建EOS密钥对和Scatter身份信息

第一步：创建EOS密钥对

在Chrome中打开并解锁Scatter插件。 打开“Key Pairs”菜单，然后单击“Create a New Key Pair”。 在下一个屏幕中，单击“ Generate Key Pair”。 生成密钥后，您应该复制并保存私钥和公钥。

 

⚠️：密钥对是在你的计算机上本地生成的。 如果未保存，你将无法访问自己的帐户。 你将无法从Scatter中取出钥匙。 所以存储和保护你的钥匙！

第二步：将Scatter身份信息关联至Bidname(EOS Mainnet)

  • 打开bidname.io，点击登陆Scatter

  • 同意身份认证请求

如果你还没有创建身份信息，请参考下面操作：

  • 打开Scatter插件，点击Identities
  • 可以选择编辑默认身份，或者创建一个新的身份
  • 编辑身份信息的Account栏，请选择EOS Mainnet 主网。主网的ID:eos:chain:aca376f206b8fc25a6ed44dbdc66547c36c6c33e3a119ffbeaef943642f0e906
  • 同样在Account栏，单击“None”以选择密钥对，然后单击“Import”。 Scatter现在将尝试查找与您选择的密钥对的公钥关联的EOS帐户。
  • 找到您的帐户后，您需要选择用于签署交易的权限。
  • 完成后，请记住单击Scatter菜单右上角的Save。恭喜！ 您刚刚在Scatter身份上设置了EOS帐户。
